1. git init
   Initializes a new Git repository.

2. git status
   Shows the current state of the working directory.

3. git add .
   Stages all changes for commit.

4. git commit -m "message"
   Saves staged changes with a message.

5. git log
   Shows commit history.

6. git branch
   Lists all branches.

7. git checkout branch_name
   Switches to another branch.

8. git pull origin main
   Fetches and merges latest changes from remote.

9. git push origin main
   Pushes commits to GitHub.

10. git clone repository_url
    Clones a remote repository.
